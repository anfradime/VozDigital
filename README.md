# 📰 VozDigital

VozDigital es un periódico digital construido con **Django**, **Vue.js** y **PostgreSQL**, diseñado para ser escalable y modular.

## 🚀 Tecnologías
- Django (Backend)
- PostgreSQL (Base de datos)
- Vue.js (Frontend)

## 🔧 Estructura del Proyecto
VozDigital/

    ├── articulos/ │
    ├── env/ │ 
    └── usuarios/

    └── vozdigital/ │ 
  ├── frontend/
